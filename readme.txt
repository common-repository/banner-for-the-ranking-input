== Description ==

This plug-in is the one only for Japanese. 

When the article is contributed, this plug-in offers the function to input the banner easily. 

In this plug-in It, the function to input the management function and the function to register the banner are provided. 

このプラグインは、
記事の投稿時に、
簡単にバナーを入力する機能を提供します。 

このプラグインは、
バナーを登録する管理機能と、
簡単にバナーを入力する機能があります。

== Installation ==

1. Upload `Directory named banner-for-the-ranking-input` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
